**Kitchen Rush** is a fast-paced, time-management cooking game developed with Unity. Players must frantically prepare and serve meals to demanding customers under intense pressure, managing ingredients, cooking stations, and plate delivery to maximize tips and clear stages.

## ğŸš€ Getting Started

Follow these steps to get a local copy of the project up and running for development or testing.

### Prerequisites

You need the following software installed:

* **Unity Hub:** Download the Unity Hub from the official website.

* **Unity Editor:** The project uses **Unity 2022.3 LTS**. Please ensure you have this specific version installed via Unity Hub.

### Installation

1. **Clone the repository:**

git clone https://github.com/Muhammad0149/KitchenRush.git


2. **Open in Unity Hub:**

* Open **Unity Hub**.

* Click **"Add"** and navigate to the `Kitchen Rush` folder you just cloned.

* Select the folder to add the project to your list.

3. **Launch the Project:**

* Click the project name in Unity Hub to open it in the Unity Editor.

4. **Run the Game:**

* The main scene is located at `Assets/Scenes/MainGame.unity` (or similar).

* Open the scene and press the **Play** button in the Unity Editor.
